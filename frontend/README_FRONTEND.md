# Mexi Events Frontend

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173) in your browser.

## Backend Connection

The frontend expects the PHP backend to be running at `http://localhost/mexi-events/api/get_events.php`.
Ensure XAMPP Apache is running and serving the `mexi-events` folder from `htdocs`.
